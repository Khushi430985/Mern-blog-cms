# Content Management System (CMS)

This is a full-stack Content Management System built using React.js, Node.js, and Express.js.

## Features
- User-friendly frontend built with React.js
- RESTful APIs for backend operations
- CRUD functionality for managing content
- Database integration for storing data

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB / PostgreSQL

## Purpose
This project was developed to understand full-stack development, frontend-backend integration, and REST API handling.
